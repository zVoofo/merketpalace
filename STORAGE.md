# Хранение файлов

## Ничего настраивать не нужно

Фото, видео и файлы в чате **сохраняются в PostgreSQL** на Render.

- Сервисные аккаунты Яндекса **не нужны**
- Cloudinary **не нужен**
- Файлы **не пропадают** после перезапуска сервера

Лимит: **50 МБ** на один файл.

---

## Как это работает

Загрузили фото в чат или объявление → файл попадает в базу → открывается по ссылке `/files/.../`

Фото в чате открывается **сразу на сайте** (лайтбокс), без скачивания.

---

## Опционально: Яндекс Object Storage

Если база станет слишком большой, можно подключить S3 (см. переменные в `DEPLOY.md`).  
Для обычного использования это **не обязательно**.
